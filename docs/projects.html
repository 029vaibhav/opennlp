Projects

  
 
Grok  

natural language processing library written in Java which provides implementations for most of the interfaces defined in the opennlp.common package.
Maxent  

mature Java package for training and using maximum entropy models.
Leo  

project to provide an architecture for defining XML specifications of grammars for different natural language parsing systems and tools for converting grammars automatically between those systems
Hylo  

Sub-project of OpenNLP for working with Hybrid Logic
Dictionary  

Sub-project of OpenNLP for working with Dictionaries, with an initial implementation using WordNet 1.7

If you are working on open source natural language software or wish to start a project and are interested in joining OpenNLP, let us know about it by either posting to the OpenNLP forum or sending mail to Jason Baldridge (jmb@cogsci.ed.ac.uk). The only requirement is that the software is related to NLP and is distributed under an open source license. You are otherwise free to use your favorite programming languages, theories, methodologies, operating systems, etc.
